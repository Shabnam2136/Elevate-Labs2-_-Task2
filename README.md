# Elevate-Labs2-_-Task2
Operating System Security Fundamentals

Overview

This repository contains the practical work and documentation for Task 2 of the Cyber Security Internship, focusing on Operating System Security Fundamentals using Kali Linux.

What Was Done

Explored Linux user accounts and access control

Understood root vs normal user privileges

Learned and applied Linux file permissions (chmod, chown)

Monitored running processes and services

Disabled unnecessary services to reduce attack surface

Enabled and configured firewall (UFW)

Studied OS hardening best practices

Tools Used

Kali Linux (Virtual Machine)

Linux Terminal Commands

Outcome

This task helped in understanding OS-level security mechanisms, Linux hardening techniques, and how securing the operating system reduces security risks.
